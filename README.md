## Quick Start
### 1. Setup Environment
```shell
conda creata -n MUSE python=3.12
conda activate MUSE
pip install -r requirements.txt
playwright install chromium
playwright install-deps chromium
```
### 2. Run Local Demo
```shell
python demo.py
```
